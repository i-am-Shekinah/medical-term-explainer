# 🩺 Medical Term Explainer

🔗 Live App: https://shekinahcodes-medical-term-explainer.hf.space

---

## 🚀 Overview

**Medical Term Explainer** is an AI-powered web app that translates complex medical terminology into simple, easy-to-understand explanations.

It is designed to help:
- Patients better understand medical reports
- Students learning healthcare concepts
- Non-medical users confused by clinical jargon

---

## ✨ Features

- 🧠 Simple, human-friendly explanations of medical terms  
- 📖 Structured breakdown (definition, context, guidance)  
- 💬 Natural language interface powered by LLMs  
- ⚡ Fast Gradio-based UI  
- ☁️ Fully deployed on Hugging Face Spaces  

---

## 🛠️ Tech Stack

- Python 🐍
- Gradio 🎨
- LangChain 🧩
- Google Gemini / LLM backend 🤖
- Hugging Face Spaces 🚀

---

## 📦 How It Works

1. User enters a medical term (e.g. "Hypertension")
2. The request is sent to a language model
3. The model generates:
   - Simple definition
   - Plain-language explanation
   - Real-world context
   - Suggested questions for a doctor
4. Response is displayed in a structured format

---

## 🎯 Example

**Input:**
Myocardial Infarction



**Output:**
- One-line definition  
- Simple explanation (layman terms)  
- When it is used or diagnosed  
- Questions to ask your doctor
